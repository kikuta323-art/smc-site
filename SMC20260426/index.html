Daily Shalom は、毎日のデボーションです。

聖書のみことばと、短い黙想と祈り。それだけを、心を込めて、あなたに届けます。

朝の光が、闇を退けるように。一杯の水が、渇いた心を潤すように。

聖書のみことばは、あなたの一日を、静かに整えてくれます。

私たちには、ただ一つの願いがあります。それは、大切な人々へ、福音(心からの平安)を伝えること。

それが私たちの使命であり、喜びです。だから、Daily Shalom はどなたにも、無料でお届けしています。

こんなひとときを

  1. 一節のみことばに、出会う
  2. 静かに、心を整える
  3. 神様に、語りかける
  4. 自分の弱さを、預ける
  5. 平安を、受け取る

Journey·歩み

走り続けて、たどり着いた場所

✦

私の名前は、菊田 吉宏 と申します。

走り続けてきた者の、
ひとつの物語を、ここに記します。

✦

私は長く、お金に関わる仕事をしてきました。証券、生命保険、損害保険、そしてそれらを横断的に扱うIFA。

30代の頃、私は次第に「この仕事は、本当に人の役に立っているのだろうか」そんな疑問を持つようになりました。

もっと、お客様一人ひとりの人生に寄り添える形があるのではないか。そう考え、転職を決意しました。

そこで出会ったのが「Love & Trust(愛と信頼)」という理念です。その言葉に、当時の私は強く心を動かされました。

ありがたいことに、多くのお客様に支えられ、やがてマネジメントの役割も任されるようになります。

しかし、忙しい日々の中で、ある問いが消えずに残りました。

「人を大切にしたい」という思いと、「成果や収益を求められる現実」は、本当に同じ方向を向いているのだろうか。

知らず知らず、数字を優先してはいないだろうか。本当に、お客様のためになっているのだろうか。

その問いは、ふとした瞬間に静かに、しかし確かに、心に浮かび続けていました。

そんな時期に、私は聖書と出会います。当時は深く考えていませんでしたが、今振り返ると、大切な転機だったと感じています。

それまでの私は、「人生は努力次第で切り開くもの」そう信じて生きてきました。

一方で、どこかに拭えない不安もありました。成果を出しても満たされない感覚。頑張り続けることへの疲れ。

そんな中で触れた聖書の言葉が、不思議と自分自身に向けられているように感じられたのです。

「すべて重荷を負うて苦労している者は、
わたしのもとにきなさい。
あなたがたを休ませてあげよう。」

その言葉は、考えるより先に、心に静かに届きました。

私はそのとき、「生き方を見直してみたい」そう思うようになりました。

その後、一度金融の世界を離れます。

けれど、最終的に私は再び戻ってきました。今度は、立場や報酬よりも、「その人の人生にとって、本当に必要なことは何か」そこに焦点を当てた関わり方をしたいと考えたからです。

保険とは何のためにあるのか。資産運用は、人生にどう役立つのか。誠実であるとは、どういう姿勢なのか。

そうした問いを大切にしながら、クライアントファーストの姿勢を貫いてきました。

しかし、もうひとつの現実にも向き合うことになります。

それは、「本当に助けを必要としている人ほど、情報にたどり着けない」という事実でした。

時間や費用をかけられる人には、十分な説明ができます。一方で、経済的にも精神的にも余裕のない人には、同じような支援が難しい。

そのことに、次第に違和感を覚えるようになりました。

私は決めました。

報酬や効率を最優先にしない。誰もが学べる形で、お金との向き合い方を伝えていこう、と。

収入は大きく減りました。それでも、不思議と心は軽くなったのを覚えています。

そこで改めて気づいたことがあります。

人が本当に求めているのは、お金そのものではなく、安心感や、希望、そして心の平穏なのではないか。

どれだけ状況が良くなっても、心が落ち着かなければ、人は満たされません。

私自身が、そのことを体験しました。

今の私は、その平安は「自分一人で頑張ること」ではなく、自分をそのまま受け止めてくれる存在との関係の中で育まれていくものだと感じています。

シャローム・ミッション・センターは、そんな「心の平安」を分かち合うための場所です。

日々の言葉や対話を通して。小さな祈りや、励ましを通して。

人生に疲れたとき、少し立ち止まれる場所でありたいと願っています。

この活動は、ビジネスではありません。私自身が受け取ってきた恵みや学びを、必要とする方と静かに分かち合うためのものです。

もしここが、あなたの日常の中で、ほんの小さな安心や希望につながるなら。

それ以上の喜びはありません。

FAQ·よくあるご質問

よくあるご質問

✦
このサイトは、何のためにあるのですか? +

このサイトは、Daily Shalom (デイリー・シャローム)という、聖書のみことばを毎朝お届けする、ささやかな働きの場です。

「心からの平安を、大切な人々へ届けたい」という願いから生まれました。特別な目的も、組織への勧誘もありません。

宗教的な内容ですか? +

はい、キリスト教信仰に基づくみことばと祈りをお届けしています。ただし、信仰を求めることはありません。

クリスチャンの方も、そうでない方も、平安を求めるすべての方に開かれています。ご自身のペースで、自由にお読みください。

利用料金はかかりますか? +

いいえ、すべて無料です。

Daily Shalom も、サイトのすべてのコンテンツも、お代をいただくことはありません。どなたでも、ご自由にお受け取りください。

「支える」ボタンのようなものはありますか? +

このサイトは、ビジネスではありません。そのため、支援を募るボタンや料金プランは置いていません。

それでも、「何か力になりたい」と願ってくださる方は、お問い合わせフォームから、お気軽にご相談ください。

支払わなくても、すべてのコンテンツを読めますか? +

はい、もちろんです。

Daily Shalom には、無料・有料の区別はありません。すべての方が、同じみことばを受け取れます。

このサイトは、どんな方に向いていますか? +

特別に「向いている方」はいません。

ただ、もし——心が疲れたとき、ふと立ち止まりたいとき、聖書のみことばに、そっと触れてみたいとき。

そんなときに、このサイトを思い出していただけたら、それで十分です。

コミュニティに参加できますか? +

参加の形は、人それぞれです。

Daily Shalom を毎朝受け取り、ご自身のペースで歩むことも、共に祈ってほしいことを送ってくださることも、それぞれが「共に歩む」一つの形です。

お問い合わせをしたら、必ずお返事をもらえますか? +

はい、できる限りお返事させていただきます。

ただ、すぐには難しい場合もありますので、少しお時間をいただくこともあります。ご了承ください。
